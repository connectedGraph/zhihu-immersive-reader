
# 📜 知乎宣纸沉浸阅读脚本 (README)

## 📖 简介

> “一叶孤舟，一卷宣纸，一盏清茶。”

本脚本专为知乎深度阅读爱好者打造。它能一键将知乎繁杂的现代网页界面（侧边栏、推荐位、各类动态图标）瞬间瓦解，重构为极简、护眼、古色古香的**“宣纸”沉浸模式**。我们不仅是在做网页清理，更是在重塑电子时代的阅读仪式感。

---

## ✨ 核心特性

* **🎨 匠心视觉调教**
* **背景板**：采用低对比度的旧麻布色（`#E5DEC9`），模拟真实桌面质感。
* **宣纸轴**：核心内容区使用特调的宣纸本色（`#F8F4E6`），辅以极细的噪点纹理，护眼且有质感。
* **点睛色**：关键元素采用中国传统色“朱砂红”，兼具古典韵味与视觉引导。


* **⚡ 毫秒级无缝热插拔**
* 采用 **DOM 内存劫持技术**，支持随时展卷/收卷，**无需刷新页面**，完美保留阅读进度、评论区状态及滚动位置。


* **🛡️ 兼容知乎原生生态**
* **操作栏**：底部的点赞、收藏、评论操作栏被重新着色并完美嵌入纸卷末端。
* **目录功能**：精准提取知乎目录按钮，将其化身为右上角的一枚“红色印章”，不仅美观，且完整保留 React 原始点击逻辑。


* **🧹 智能垃圾清理**
* 自动识别并斩断文中广告、MCN 推广卡片、文章末端引流推荐，只留纯净文字与图像。


* **⌨️ 快捷键支持**
* 默认支持 `Ctrl + E` (Windows/Linux) 或 `Cmd + E` (macOS) 瞬间切换模式。

## 🚀 使用说明

### 方法一：控制台临时体验（极客推崇）

1. 在知乎文章/回答页面，按 `F12` 或右键点击“检查”打开开发者工具。
2. 切换到 **Console** (控制台) 面板。
3. 复制本项目中 `patch.js` 的完整代码并粘贴，按回车即可。

### 方法二：油猴脚本（懒人常驻）

1. 安装浏览器扩展 `Tampermonkey`。
2. 点击插件图标 -> **添加新脚本**。
3. 将本项目 `patch.js` 代码粘贴进去并保存。
4. 脚本已自动配置适配规则 `*://*.zhihu.com/*`，刷新页面即可生效。

### 方法三：浏览器书签法（优雅随行 - **推荐！**）

这种方法无需安装插件，只要像点击收藏夹一样点一下就能开启，非常适合追求极致轻量的用户。

1. 在浏览器书签栏**右键 -> 添加网页**（或新建书签）。
2. **名称**：写“📜 宣纸模式”
3. **网址/URL**：将本项目 `bookmarklet.txt` 中的那一段以 `javascript:` 开头的代码全部复制进去。
4. **用法**：以后在知乎看长文章时，点一下书签栏的这个“📜 宣纸模式”，立刻变身！

## 🕹️ 交互指南

| 交互动作 | 实现效果 |
| --- | --- |
| **快捷键 `Ctrl + E**` | 在“现代模式”与“宣纸模式”之间无缝切换 |
| **左下角【退出沉浸】** | 归还文章主体，恢复知乎原貌 |
| **右上角【朱砂印章】** | 唤出/关闭文章目录（仅对包含目录的文章生效） |
| **页面滚动** | 按钮采用固定定位，始终在视窗两角静候 |

## 🛠️ 技术实现

本项目采用 **非破坏性 DOM 提取策略**。通过在原位置埋下 `Placeholder` 占位符，并在进入模式时利用 `Fixed` 布局构建临时渲染层，确保了在模式切换过程中，React 组件的生命周期与事件监听不会被销毁，实现了极佳的性能表现与交互兼容性。
